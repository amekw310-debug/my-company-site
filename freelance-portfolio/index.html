About

はじめまして

あみ

A@Web Works / Webクリエイター

高校時代からWebデザイナーに憧れ、専門学校でプログラミング言語やホームページ制作の仕組みについて学んできました。

当時は、ゼロからデザインを生み出す力が求められる時代。まわりのセンスの良さに圧倒され、自分には向いていないのではないかと感じ、一度はWebの仕事をあきらめました。

しかし、AIの登場によって状況は大きく変わりました。
頭の中にあるイメージや雰囲気を、言葉や参考イメージをもとに形にできるようになり、以前は表現しきれなかったアイデアも実現できる時代になりました。

今だからこそ、自分が本当にやりたかったWeb制作の仕事にもう一度挑戦したい。そんな思いから、AIを活用したホームページ制作を始めました。

まだAIに抵抗がある方や、信用しきれないと感じる方も少なくありません。だからこそ私は、AIにすべてを任せるのではなく、目的に合わせて適切にコントロールし、良さを最大限に引き出すパートナーとして活用していきます。

AIの力を取り入れることで、低コスト・短納期でありながら、想いが伝わるホームページ制作を目指します。また、TikTok・InstagramなどSNS運用の実務経験も活かし、ホームページ公開後の集客や採用活動までサポートいたします。

採用実務経験あり SNS運用（TikTok・Instagram） 公式LINE構築 HTML / CSS GitHub公開対応 AI活用制作
