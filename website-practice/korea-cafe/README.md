## プロジェクト概要

東京・原宿を拠点とする韓国スタイルカフェ「카페 봄（カフェ ボム）」のコーポレートサイト。  
韓国直輸入の食材・茶葉を使ったドリンク・スイーツ・フードを提供する女性向けカフェ。

| 項目 | 内容 |
|------|------|
| 店舗名 | 카페 봄（カフェ ボム） |
| 所在地 | 東京都渋谷区神宮前 4-25-11F（ダミー） |
| 営業時間 | 10:00 〜 21:00（月曜定休） |
| 電話 | 03-5678-9012（ダミー） |
| コンセプト | 春のようにあたたかく、おだやか |
| ターゲット | 韓国カフェ好きの女性・10〜30代 |
